The Watson Platform

The Watson Platform is a library of discrete cognitive components that can be pulled off the virtual shelf by everyone from entrepreneurs to corporate developers--and used in any number of applications.

Due to high demand of the IBM Watson Developer Cloud for entrepreneurs, enterprises will now be able to harness the power of the Watson API through the Watson Experience Preview. This new offering is a cloud-computing community that enables corporate developers to prototype and test new enterprise applications made with Watson.
