What is v17e?
v17e is the mobile optimized version of the v17 design system for ibm.com. It's a framework that will enable you to create a cohesive and engaging digital experience on the ibm.com domain. It's a foundation you can build on top of with extreme flexibility and efficiency. It's a set of tools and resources to help you focus on what really matters, the customer.
What's new in v17e?
Adaptive and responsive grids for desktop, tablet, and mobile screens. Other enhancements: updated jquery code, icon's have been converted to a font, IBM custom fonts, as well as new, redesigned modules.
What's new in v17e? (details)
There are a few new classes and containers in v17e that you can use to improve your page appearance for mobile devices or just generally to tweak the page. Three biggies include 1) adding the ibm-resize class to an image tag allows the image to get smaller (or larger) as needed based on window size, 2) the ibm-band class puts an edge to edge band around ibm-columns, visually breaking the space into horizontal bands and opening up the space to the left and right of the content for use, and 3) ibm-sprite lets you have an image that looks different on mouse over or down using a single image (sprite).
How do I get started? How do I get access?
