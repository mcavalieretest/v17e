Pop-Up Window Example

Secondary browsers should not be confused with pop-up windows or overlays, though all three are supported under v17.

  • Secondary browsers (also known as secondary windows) are second instances of an Internet browser window that, when the user clicks a link, opens and displays information from a separate HTML file.
  • Pop-up windows are also second instances of an Internet browser window that, when the user clicks a link, opens and displays information from a separate HTML file. However, the second instance of the browser displays a modified masthead and footer.

Secondary browsers and pop-up windows can be confusing and frustrating to users when implemented without proper care. Whenever possible, designers should use an overlay instead.
